# 贡献指南

## 提交规范

使用 Conventional Commits：

```
type(scope): description

feat(today): 添加今日页喝水追踪卡片
fix(router): 修复深链导航问题
refactor(shell): 重构底部导航状态管理
```

### Type 枚举

- `feat`: 新功能
- `fix`: 修复
- `refactor`: 重构
- `style`: 样式调整
- `docs`: 文档
- `test`: 测试
- `chore`: 构建/工具

## 分支策略

- `main`: 稳定版本
- `dev`: 开发分支
- `feat/*`: 功能分支
- `fix/*`: 修复分支
