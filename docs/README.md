# Luminous 项目文档中心 (Docs)

本项目所有的说明、规划、接口对齐以及历史学习资料均归档于此。
请以本目录作为所有架构规范和长线计划的依据。

## 目录结构

*   **Roadshow/**：项目早期和近期用于评奖、路演的各种参赛资料、PPT及大纲。（**暂留，请勿删除**）
*   **study/**：项目初期的架构分层、模块学习与排障文档记录。
*   **lib-docs/**：前后端接口对齐清单与部署配置手册。
*   **oot/**：早期项目根目录存放的老旧说明类文档与检查清单。

## 核心长线规划（重要）

*   **LUMINOUS_HEALTH_COPILOT_VISION.md**：全终端演进与商业路演愿景（项目的北极星指标与核心重构指导）。
*   **OPTIMIZATION_AND_NEST_MIGRATION_PLAN.md**：当前 Flutter 架构的局部优化以及后端向 NestJS 迁移的落地计划。

## 其他

*   CURRENT_REFACTOR_STATUS.txt：早期开发中的某次重构遗留状态备份。
*   privacy-policy.txt：项目的隐私政策文本档案。
